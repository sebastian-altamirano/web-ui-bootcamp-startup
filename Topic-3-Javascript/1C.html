isPalindrome interface

Enter a word or a phrase to know if it's a palindrome (a string that reads the same backwards as forwards). E.g: "Eva, can I see bees in a cave?"

getDayAfter interface

Specify a day and a number to know what day it's after that number of days have passed from the specified day. E.g: "Monday" and "4" return "Friday".

4 days after Monday it'll be Friday
